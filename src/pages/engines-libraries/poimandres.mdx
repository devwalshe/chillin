# ⚫️ Poimandres Libraries

[Poimandres](https://github.com/pmndrs) is an open-source developer collective, which is an active community developing many tools for Three.js, making 3D in React, and a lot more (you might know them from Zustand or react-spring). Let’s take a closer look at the tools they develop.
